# Cloudflare R2 L3 Store

AeroCache supports an optional L3 store backed by Cloudflare R2 through its S3-compatible API.

L3 is disabled by default.

## Purpose

L3 is for larger or colder cache entries that should survive process restarts and Valkey loss.
